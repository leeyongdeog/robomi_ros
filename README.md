## usb cam 2대를 pi 연결 작동 요령 (결과물 실행 위주)

```
roscore   # pc ip123에서 실행
```

```
roslaunch usb_cam usb_cam_0-test.launch   # pi ip128에서 실행
```

```
roslaunch usb_cam usb_cam_2-test.launch   # pi ip128에서 실행
```
